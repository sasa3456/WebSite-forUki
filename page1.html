How to install and compile a program on Uki?

1. Unpack the zip archive. 2. Move the FORGER and MinGW folders to disk C:\ 3. Create a variable in the Windows environment variables and enter this line "C:\MinGW\bin " 4. Write the code itself to the unitled.uki file 5. Run the compile application Uki.exe 6. open the console and enter this line "gcc source.c -o Uki.exe "! Uki.exe this is your program

Data types of variables?

1. int can store numbers up to 32000 2. double and float can store floating point numbers 3. string can store a string 4. char can store a single character

Blocks?


The block function declares that you have created a block. Blocks can be connected to each other via the command $connection [the name of the block with which you want to associate]. With the $exit command, we show which block is the main one

Out() and entry() functions

out() - output to the console. entry() - input to the console

Cycles

Parametric "for" loop:
int k; 
int sum = 0;
for(int i=1; i<=k; i++) 
{
sum = sum + i; 
}

A loop with a "while" precondition:
int k;
int i = 1;
int sum = 0;
while (i <= k) 
{
sum = sum + i; 
i++; 
}

If, self and else checks

if:
int i = 6;
if ( i == 6 )
   out(i);

self:
int i = 6;
self ( i == 6 )
out(i);

else:
int i = 6;
else
   out(i);

Comments

Comments in Uki are written //

Increments and decrements


Increments are written [any variable]++. Decrements are written [any variable]--
